# Chapter 10 - Loops

This chapter teaches repeated execution with `for`, `while`, `do...while`, and collection loops. It also includes common practice problems like FizzBuzz and triangle classification.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `For_loop.js`, `For_Loop2.js`, `for_loops_js` | `for` loops | Repeats code using counters. |
| `While.js` | `while` loop | Runs while a condition remains true. |
| `Do_While.js`, `do_while_loop.js` | `do...while` | Runs once before checking the condition. |
| `For_OF_IN_EACH.js` | Collection loops | Practices `for...of`, `for...in`, and `forEach`. |
| `FizzBuzz_Test.js` | FizzBuzz | Prints values based on divisibility rules. |
| `HomeWork_Triangle_Classifier_js` | Triangle classifier | Uses conditions and repeated practice logic. |
| `interview.js`, `IQ.js` | Interview practice | Output-based loop questions. |

## Concepts Covered

- `for`, `while`, and `do...while`
- Loop counters and conditions
- Iterating arrays/objects
- Repeated decision logic
- Common coding interview exercises

## How To Run

```bash
node Chapter_10_Loops/For_loop.js
node Chapter_10_Loops/FizzBuzz_Test.js
node Chapter_10_Loops/While.js
```

## Takeaway

Loops reduce repeated code and help process lists, counters, and repeated test scenarios.
