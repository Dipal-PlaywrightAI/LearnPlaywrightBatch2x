# Chapter 15 - 2D Arrays

This chapter explains arrays inside arrays. The practicals show matrix-style data, nested access, functions with 2D arrays, patterns, and real examples.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `Arrays_2D.js`, `array_2d.js` | 2D array basics | Creates nested arrays and accesses rows/columns. |
| `Array_2D_Fun.js` | Functions with 2D arrays | Processes nested array data through functions. |
| `Pattern.js` | Pattern printing | Uses nested loops to print patterns. |
| `Real_Ex.js` | Real example | Applies 2D array thinking to practical data. |

## Concepts Covered

- Nested arrays
- Row and column access
- Nested loops
- Pattern logic
- Matrix-like data handling

## How To Run

```bash
node Chapter_15_2D_Array/Arrays_2D.js
node Chapter_15_2D_Array/Pattern.js
```

## Takeaway

2D arrays help represent table-like data, grids, matrices, and grouped test data.
