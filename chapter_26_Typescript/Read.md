# Chapter 26 - TypeScript

This chapter introduces TypeScript basics and type-safe JavaScript development. The practicals include hello world, typed variables, functions, objects, and interview questions.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `HelloWorld.ts` | First TypeScript file | Runs a simple TypeScript example. |
| `TS_Part1.ts` | TypeScript basics | Introduces basic types and syntax. |
| `TS_Part2.ts` | More type practice | Continues typed variables/functions. |
| `TS_Part3.ts` | Advanced basics | Adds more TypeScript examples. |
| `Ts_Part4.ts` | Continued practice | Extends the TypeScript concept set. |
| `IQ.ts`, `IQ2.ts` | Interview practice | TypeScript concept/output questions. |

## Concepts Covered

- TypeScript files with `.ts`
- Type annotations
- Typed variables and functions
- Compile-time checking
- Interview-style TypeScript questions

## How To Run

Use `ts-node` if available:

```bash
npx ts-node chapter_26_Typescript/HelloWorld.ts
```

Or compile with TypeScript:

```bash
npx tsc chapter_26_Typescript/HelloWorld.ts
node chapter_26_Typescript/HelloWorld.js
```

## Takeaway

TypeScript helps catch type mistakes earlier and is commonly used with Playwright test automation.
