# Chapter 11 - Arrays

This chapter covers JavaScript arrays from creation to common methods. The practicals show how to access, update, search, sort, combine, and transform array data.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `Basic_Array.js`, `Arrays.js` | Array basics | Creates arrays and stores multiple values. |
| `Access_Array.js` | Accessing values | Reads array items by index. |
| `Arrays_push_pop_sift_unshift.js` | Add/remove items | Uses push, pop, shift, and unshift. |
| `array_splice.js` | Splice | Adds/removes items at specific positions. |
| `Slice_Combining.js`, `Concat_array.js` | Slice/concat | Copies and combines arrays. |
| `Searching.js`, `Array_Checking.js` | Search/check | Finds values and checks array conditions. |
| `Sorting_Arrays.js` | Sorting | Sorts array values. |
| `Every_some.js` | `every` and `some` | Checks all/some items against a condition. |
| `Iterate.js` | Iteration | Loops through array values. |
| `Transform_Array.js`, `Array_Method_Practice.js` | Transform methods | Practices methods like map/filter-style transformations. |
| `Real_Example.js` | Real example | Applies arrays to practical data handling. |

## Concepts Covered

- Array creation and indexing
- Mutable array methods
- Searching and checking arrays
- Sorting and combining
- Iteration and transformation

## How To Run

```bash
node Chapter_11_Array/Basic_Array.js
node Chapter_11_Array/Array_Method_Practice.js
node Chapter_11_Array/Real_Example.js
```

## Takeaway

Arrays are essential for handling lists of data, especially in automation where test data and results often come as collections.
