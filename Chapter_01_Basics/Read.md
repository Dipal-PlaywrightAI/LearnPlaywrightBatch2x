# Chapter 01 - Basics

This chapter starts the JavaScript setup and first programs. The practicals focus on writing simple JS files, running code, checking the environment, and understanding basic output.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `01_Basic.js` | First JavaScript basics | Introduces simple statements and console output. |
| `02_JS.js` | JavaScript syntax practice | Continues basic JS execution and simple examples. |
| `03_JS_Verify_Setup.js` | Setup verification | Confirms Node.js/JavaScript code is running correctly. |
| `04_HotCode` | Extra quick practice | Small hands-on code practice for the basics chapter. |

## Concepts Covered

- Creating and running JavaScript files
- Using `console.log()`
- Verifying the local setup
- Getting comfortable with basic syntax

## How To Run

```bash
node Chapter_01_Basics/01_Basic.js
node Chapter_01_Basics/02_JS.js
node Chapter_01_Basics/03_JS_Verify_Setup.js
```

## Takeaway

After this chapter, you should be able to create a JavaScript file, run it with Node.js, and understand the first level of program output.
