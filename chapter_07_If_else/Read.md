# Chapter 07 - If Else

This chapter teaches decision making with `if`, `else if`, and `else`. The examples include simple checks, real-world conditions, and interview questions.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `IF_ESLE.js`, `IE.js` | Basic if/else | Runs simple true/false conditions. |
| `If_elseif_else.js` | Multiple conditions | Checks several branches in order. |
| `IF_ELSE_real.js`, `REAL_IF_ELSE.js` | Real examples | Applies conditions to practical scenarios. |
| `Grade_Calc.js` | Grade calculator | Converts marks/score into grade output. |
| `LEAP_YEAR.js` | Leap year logic | Checks whether a year is leap year. |
| `API_IF_ELSE.js` | API-style decisions | Uses conditions for response/status logic. |
| `IQ.js`, `IQ_EVEN_ODD.js`, `IQ_IF_ELSE.js` | Interview questions | Practices expected output and logic questions. |

## Concepts Covered

- `if`, `else if`, `else`
- Boolean conditions
- Even/odd logic
- Range checks
- Real decision flow
- Interview-style branching questions

## How To Run

```bash
node chapter_07_If_else/Grade_Calc.js
node chapter_07_If_else/LEAP_YEAR.js
node chapter_07_If_else/IQ_EVEN_ODD.js
```

## Takeaway

Use if/else when the program must choose different actions based on conditions.
