# Chapter 18 - Async And Await

This chapter teaches `async`/`await`, sequential execution, parallel execution, and practical async flow. These ideas are important for Playwright automation.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `Async.js`, `Async_Ex.js`, `Aysnc_P2.js` | Async basics | Creates async functions and awaits results. |
| `Seq.js` | Sequential execution | Runs async steps one after another. |
| `Parallel.js` | Parallel execution | Runs independent async work together. |
| `ACLogin.js` | Login-style async flow | Applies async/await to an automation-like flow. |
| `_PyODom.js` | Pyramid of doom comparison | Relates async/await back to callback nesting. |
| `IQ.js` | Interview practice | Async/await output and behavior questions. |

## Concepts Covered

- `async` functions
- `await`
- Promise handling with cleaner syntax
- Sequential vs parallel async execution
- Automation-style async steps

## How To Run

```bash
node Chapter_18_Acyncamdawait/Async.js
node Chapter_18_Acyncamdawait/Seq.js
node Chapter_18_Acyncamdawait/Parallel.js
```

## Takeaway

`async`/`await` is the preferred style for readable asynchronous automation code.
