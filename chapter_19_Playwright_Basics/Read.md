# Chapter 19 - Playwright Basics

This chapter starts Playwright test automation. The practicals are TypeScript spec files inside the `tests` folder.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `tests/example.spec.ts` | First Playwright spec | Demonstrates the basic Playwright test structure. |
| `tests/Exersize1.spec.ts` | Exercise 1 | Practices browser/page automation steps. |
| `tests/exersize2.spec.ts` | Exercise 2 | Continues Playwright locator/action/assertion practice. |

## Concepts Covered

- Playwright test files
- `test()` structure
- Browser/page usage
- Locators, actions, and assertions
- Running tests from the command line

## How To Run

```bash
npx playwright test chapter_19_Playwright_Basics/tests
```

To view the report after a run:

```bash
npx playwright show-report
```

## Takeaway

This chapter connects JavaScript/TypeScript fundamentals with browser automation using Playwright.
