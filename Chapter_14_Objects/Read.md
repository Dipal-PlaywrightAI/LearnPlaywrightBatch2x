# Chapter 14 - Objects

This chapter covers JavaScript objects, object creation, properties, methods, references, spread, destructuring, getters/setters, and real object examples.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `objects.js`, `Objects2.js`, `object_example.js` | Object basics | Creates objects and accesses properties. |
| `Object.creation.js` | Object creation | Shows different ways to create objects. |
| `Objects_Funcations.js` | Object methods | Stores functions inside objects. |
| `Objects_REal.js`, `Obj_REAL.js` | Real examples | Applies objects to realistic data. |
| `Let_const_obj.js` | Object with `let`/`const` | Shows mutation behavior with object bindings. |
| `Primitive_Ref.js` | Primitive vs reference | Compares value copy and reference copy behavior. |
| `Object_Spred.js` | Object spread | Copies/merges objects using spread syntax. |
| `Object_Decon.js` | Destructuring | Extracts object properties into variables. |
| `Object_Get_Set.js` | Getter/setter | Uses controlled access to object properties. |
| `IQ.js`, `IQ1.js` | Interview practice | Object output and reference questions. |

## Concepts Covered

- Object literals
- Properties and methods
- Dot and bracket access
- Primitive vs reference values
- Spread and destructuring
- Getters and setters

## How To Run

```bash
node Chapter_14_Objects/objects.js
node Chapter_14_Objects/Object_Spred.js
node Chapter_14_Objects/Primitive_Ref.js
```

## Takeaway

Objects model real data. They are heavily used in test data, API responses, configuration, and page object style code.
