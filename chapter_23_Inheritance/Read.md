# Chapter 23 - Inheritance

This chapter teaches inheritance in JavaScript classes. The practicals cover single, multilevel, hierarchical, and attempted multiple inheritance examples.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `Single_Inheritance.js`, `Single_Inheritance_Example.js` | Single inheritance | One child class extends one parent class. |
| `Single_con.js` | Constructor inheritance | Uses parent/child constructors. |
| `Multiple_Level_Inheritance.js` | Multilevel inheritance | Builds a chain of inherited classes. |
| `Hierarchial_Inheritance.js` | Hierarchical inheritance | Multiple child classes inherit from one parent. |
| `Multiple_Inheritance.js` | Multiple inheritance discussion | Demonstrates the concept/limitation in JavaScript. |
| `ReaL_Example.js` | Real example | Applies inheritance to practical modeling. |
| `Iq.js`, `IQ2.js` | Interview practice | Output and concept questions. |

## Concepts Covered

- `extends`
- Parent and child classes
- Constructor chaining
- Method reuse
- Multilevel and hierarchical inheritance
- Inheritance interview questions

## How To Run

```bash
node chapter_23_Inheritance/Single_Inheritance.js
node chapter_23_Inheritance/Multiple_Level_Inheritance.js
node chapter_23_Inheritance/Hierarchial_Inheritance.js
```

## Takeaway

Inheritance lets child classes reuse and specialize parent class behavior.
