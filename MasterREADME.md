## Setup

Install dependencies from the repository root:

```bash
npm install
```

Main dependencies/dev dependencies include:

- `playwright`
- `@playwright/test`
- `typescript`
- `ts-node`
- `@types/node`

## Running JavaScript Examples

Run any `.js` file with Node.js from the project root:

```bash
node Chapter_01_Basics/01_Basic.js
node Chapter_12_Functions/Closure.js
node chapter_23_Inheritance/Single_Inheritance.js
```

Some input examples may need `prompt-sync` installed or may need code uncommented before running.

## Running TypeScript Examples

Run a TypeScript file with the project script:

```bash
npm run ts -- chapter_26_Typescript/HelloWorld.ts
npm run ts -- chapter_27_TypeScript_Interface/Interface.ts
```

Or compile with TypeScript:

```bash
npx tsc chapter_26_Typescript/HelloWorld.ts
node chapter_26_Typescript/HelloWorld.js
```

## Running Playwright

Install Playwright browsers if they are not installed:

```bash
npx playwright install
```

The current Playwright practical files are in:

```text
chapter_19_Playwright_Basics/tests/
```

The current `playwright.config.js` uses `testDir: './tests'`. If you want `npm test` to find the Chapter 19 specs directly, update `testDir` to the chapter test folder or move/copy specs into a root `tests/` folder.
