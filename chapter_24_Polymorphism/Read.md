# Chapter 24 - Polymorphism

This chapter introduces polymorphism through method overriding.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `Method_Overriding.js` | Method overriding | A child class provides its own version of a parent method. |

## Concepts Covered

- Polymorphism
- Method overriding
- Same method name, different behavior
- Runtime behavior based on object type

## How To Run

```bash
node chapter_24_Polymorphism/Method_Overriding.js
```

## Takeaway

Polymorphism allows related classes to share method names while providing different behavior.
