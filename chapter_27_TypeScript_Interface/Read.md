# Chapter 27 - TypeScript Interface

This chapter focuses on TypeScript interfaces and readonly properties.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `Interface.ts` | Interface basics | Defines object shapes using an interface. |
| `INterface_2.ts` | Interface practice | Continues interface examples and usage. |
| `IF_READONLY.ts` | Readonly interface property | Prevents reassignment of selected fields. |
| `IF_READONLY2.ts` | More readonly practice | Adds extra readonly interface examples. |

## Concepts Covered

- Interface declaration
- Object shape checking
- Optional/required property thinking
- `readonly` properties
- Type-safe object contracts

## How To Run

```bash
npx ts-node chapter_27_TypeScript_Interface/Interface.ts
npx ts-node chapter_27_TypeScript_Interface/IF_READONLY.ts
```

## Takeaway

Interfaces make object structure clear and help TypeScript catch incorrect data shapes before runtime.
