# Chapter 12 - Functions

This chapter explains JavaScript functions in depth: declarations, expressions, parameters, return values, arrow functions, scope, closures, IIFE, pure functions, rest/spread, and higher-order functions.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `Functions.js` | Function basics | Introduces defining and calling functions. |
| `Type1_Fn_Basic_Functions.js` | No parameter/no return | Basic function type. |
| `Type2_Fn_With_Param_No_Return.js` | Parameter/no return | Passes values into functions. |
| `Type3_Fn_without_Param_Return_Type.js` | Return without parameter | Returns a value from inside a function. |
| `Type4_Fn_With_Param_With_Return.js` | Parameter with return | Uses input and returns output. |
| `Fn_Expression.js` | Function expression | Stores a function in a variable. |
| `Arrow_Fn.js`, `Arrow_Fn_REAL.js` | Arrow functions | Writes compact function syntax and real examples. |
| `Default_Param_Fn.js` | Default parameters | Provides fallback parameter values. |
| `Rest_Param_Fn.js` | Rest parameters | Accepts multiple values as an array. |
| `Template_literal.js` | Template output | Builds dynamic strings in function examples. |
| `Scope._Fn.js` | Scope | Shows local/global scope behavior. |
| `Closure.js`, `API_REAL_Closure.js` | Closures | Shows functions remembering outer variables. |
| `IIFE.js` | IIFE | Runs a function immediately. |
| `Pure_Fn.js` | Pure functions | Demonstrates predictable function output. |
| `Higher_Order_Fn.js` | Higher-order functions | Passes/returns functions. |
| `Spead_IQ.js`, `IQ.js`, `IQ!.js`, `IQ1.js` | Interview practice | Function output and concept questions. |

## Concepts Covered

- Function declarations and expressions
- Parameters and return values
- Arrow functions
- Scope and closures
- IIFE
- Pure functions
- Rest and spread
- Higher-order functions

## How To Run

```bash
node Chapter_12_Functions/Functions.js
node Chapter_12_Functions/Closure.js
node Chapter_12_Functions/Higher_Order_Fn.js
```

## Takeaway

Functions organize reusable logic. This chapter is one of the most important foundations for automation, callbacks, promises, and test helpers.
