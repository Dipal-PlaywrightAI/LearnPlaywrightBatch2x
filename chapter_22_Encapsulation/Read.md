# Chapter 22 - Encapsulation

This chapter explains encapsulation: keeping data protected and exposing controlled methods to work with it.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `Encapsulation.js` | Encapsulation basics | Demonstrates hiding internal details behind methods. |
| `Ecap_Bank.js` | Bank example | Protects account-style data and controls updates. |
| `Ecap_car.js` | Car example | Uses object/class behavior to manage car data. |
| `Real_Example.js` | Real example | Applies encapsulation to a practical scenario. |

## Concepts Covered

- Data hiding
- Controlled access through methods
- Preventing direct unwanted changes
- Real-world class design

## How To Run

```bash
node chapter_22_Encapsulation/Encapsulation.js
node chapter_22_Encapsulation/Ecap_Bank.js
```

## Takeaway

Encapsulation keeps object data safer and makes class behavior easier to control.
