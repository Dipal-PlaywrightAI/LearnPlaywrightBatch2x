# LearnPlaywrightBatch2x

A personal JavaScript learning repository built around early Playwright preparation.

This project captures the first chapters of a learning path that focuses on JavaScript fundamentals, development setup checks, and editor productivity. It is organized by chapter so each topic is easy to explore and run.

## Why this repo

- Practice JavaScript with clear, chapter-based examples
- Verify your Node.js environment and platform details
- Build an understanding of identifiers, values, and comment syntax
- Keep learning progress organized as the project grows

## What is available today

### Chapter 1 вЂ” Basics
- `01_Basic.js`: first JavaScript output and console usage
- `02_JS.js`: variable declarations, loops, and simple logic
- `03_JS_Verify_Setup.js`: checks Node version, platform, and architecture
- `04_HotCode.js`: explores hot code behavior and performance patterns

### Chapter 2 вЂ” Core JavaScript
- `05_JS_Basics.js`: deeper JavaScript examples and behavior notes

### Chapter 3 вЂ” Identifiers & Literals
- `06_Identifier_Rules.js`: valid identifier names and naming rules
- `07_Identifier_Part2.js`: more identifier examples and edge cases
- `08_Comments.js`: comment formats and best practices
- `js_identifier_rules.js`: helper examples for naming rules
- `VS_Code_keyboard_shortcut_windows.md`: editor shortcuts for faster work

### Chapter 4 вЂ” JavaScript Concepts Continued
- `09_var_let_Const.js`: differences between `var`, `let`, and `const`
- `10_var_explain.js`: detailed `var` behavior and hoisting examples
- `11_let_explain.js`: `let` scoping and temporal dead zone
- `12_const_explain.js`: constant declarations and immutability notes
- `13_hosting_var.js`: hoisting behavior for `var` declarations

## Getting started

### Prerequisites
- Node.js 18 or higher
- npm (bundled with Node.js)
- Git
- VS Code or another code editor

### Run the examples

```bash
cd LearnPlaywrightBatch2x
node Chapter_01_Basics/01_Basic.js
node Chapter_01_Basics/03_JS_Verify_Setup.js
```

### Expected output

For `01_Basic.js`, you should see:

```bash
Hello The Testing Academy
```

## How to use this repo

1. Open the folder in VS Code.
2. Read each chapter file and review the comments.
3. Run the examples with Node.js.
4. Modify code to test your understanding.
5. Add new files as you continue learning.
