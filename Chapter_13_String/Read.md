# Chapter 13 - Strings

This chapter practices JavaScript string creation, properties, conversion, searching, substrings, and transformation methods.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `String.js` | String basics | Creates and prints string values. |
| `String_Properties.js` | String properties | Uses properties like string length. |
| `String_Conversion.js` | Conversion | Converts values to strings and strings to other types. |
| `Search_checking.js` | Search/check | Checks whether text contains or matches values. |
| `Substrings.js` | Substrings | Extracts parts of a string. |
| `Transform.js` | Transformations | Changes casing, trims, and modifies string values. |

## Concepts Covered

- String literals
- Length and indexing
- Searching text
- Extracting substrings
- Transforming string data

## How To Run

```bash
node Chapter_13_String/String.js
node Chapter_13_String/Search_checking.js
node Chapter_13_String/Transform.js
```

## Takeaway

String handling is useful for test validation, selectors, messages, URLs, and API data checks.
