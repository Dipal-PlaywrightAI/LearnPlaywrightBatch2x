# Chapter 06 - Operators And Equality

This chapter covers JavaScript operators, including arithmetic, comparison, strict/loose equality, logical operators, increments, ternary checks, and type checks.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `Oprators.js` | Operator overview | Introduces common JavaScript operators. |
| `Arethmetic_oprtor.js` | Arithmetic | Uses `+`, `-`, `*`, `/`, and related math operations. |
| `Modulus_OP.js` | Modulus | Uses `%` for remainder checks. |
| `Exponentioal_OP.js` | Exponentiation | Uses `**` for powers. |
| `Compund_Operator.js` | Compound assignment | Practices `+=`, `-=`, and similar operators. |
| `Incre_Decre_Op.js`, `Post_Increment.js` | Increment/decrement | Shows pre/post increment behavior. |
| `Comparsion_OP.js` | Comparisons | Uses `>`, `<`, `>=`, `<=`. |
| `Comparsion_Strict_loose.js` | `==` vs `===` | Compares loose and strict equality. |
| `Confusing_Comparsion.js` | Tricky comparisons | Shows examples where type conversion can surprise you. |
| `Logical_Op.js` | Logical operators | Uses `&&`, `||`, and `!`. |
| `Ternary_Op.js` | Ternary operator | Writes compact conditional expressions. |
| `Type_Op.js` | Type operator | Uses `typeof` to inspect values. |
| `Null_Op.js` | Nullish logic | Practices null-related checks. |
| `String_Con_Op.js` | String concatenation | Combines strings and values. |
| `Advance_ID_.js`, `IQ_*.js` | Interview practice | Extra questions for equality and operator output. |

## Concepts Covered

- Arithmetic and assignment operators
- Strict equality `===` vs loose equality `==`
- Type conversion during comparisons
- Logical and ternary operators
- Increment/decrement output questions

Prefer strict equality `===` in real code, and practice operator output questions because they appear often in JavaScript interviews.
