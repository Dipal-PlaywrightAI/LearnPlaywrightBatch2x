# Chapter 21 - Class And Object

This chapter teaches JavaScript classes and objects. The practicals cover class creation, constructors, static members, access style, and real examples.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `Class_Object.js` | Class/object basics | Creates a class and object instances. |
| `Constructor.js` | Constructor | Initializes object data when creating instances. |
| `Private_Public.js` | Access style | Demonstrates public/private-style member usage. |
| `Static.js` | Static properties/methods | Uses class-level members. |
| `Static_Function.js` | Static function | Calls a function directly on the class. |
| `RealExample.js` | Real example | Applies class/object code to a realistic scenario. |
| `IQ.js` | Interview practice | Class output and concept questions. |

## Concepts Covered

- Classes and objects
- Constructors
- Instance members
- Static members
- Public/private concepts
- Real-world modeling

## How To Run

```bash
node Chapter_21_Class_Object/Class_Object.js
node Chapter_21_Class_Object/Constructor.js
node Chapter_21_Class_Object/Static.js
```

## Takeaway

Classes help group data and behavior together, making larger programs easier to structure.
