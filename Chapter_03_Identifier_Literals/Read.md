# Chapter 03 - Identifiers And Comments

This chapter explains JavaScript identifier rules, naming style, comments, and some useful VS Code shortcuts for writing code faster.

## Practical Files

| File | Practical | What happens |
| --- | --- | --- |
| `06_Identifier_Rules.js` | Identifier rules | Shows valid and invalid variable/function naming rules. |
| `07_Identifier_Part2.js` | More identifier practice | Continues naming examples and common mistakes. |
| `08_Comments.js` | Comments | Demonstrates single-line and multi-line comments. |
| `js_identifier_rules.js` | Identifier recap | Extra examples for JavaScript naming rules. |
| `VS_Code_keyboard_shortcut_windows.md` | VS Code shortcuts | Lists useful Windows keyboard shortcuts for coding. |

## Concepts Covered

- What identifiers are
- Valid variable and function names
- Reserved words and naming restrictions
- Comments for notes and disabling code
- Editor productivity shortcuts

## How To Run

```bash
node Chapter_03_Identifier_Literals/06_Identifier_Rules.js
node Chapter_03_Identifier_Literals/07_Identifier_Part2.js
node Chapter_03_Identifier_Literals/08_Comments.js
```

## Takeaway

You learn how to name things correctly in JavaScript and how comments help explain or temporarily skip code.
